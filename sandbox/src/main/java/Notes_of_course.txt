Переменные и значения. 
У любой переменной есть значение. 
Значение -- это данные, которым равна переменная в данный момент выаолненения программы. 
Значение может быть не только у переменных, но и у любого другого параметра, например  в строке "Hello world", "Hello world" -- это тоже значениние. Значения бывают литеральные (явные) и вычисляемые. 
Переменные -- идентификатор или имя, которое ссылается на значение. Переменная всегда определена в момент выполнения программы в Java. 

Типы переменных;
Java Язык строгой типизации или строгого котроля типов ( то есть когда проверяются типы сразу на этапе компиляции, а не выполнения программы); 
int --  указатель (тип) на целочисленные значения; int variableOne = 3;
double --  указатель (тип) на число с плавающей точкой; double variableTen = 21; // это тоже самое, что double variableTen = 21.0; и обработка данных будет в числах с плавающей точкой и результат тоже будет в числах с плавающей точкой
String --  указатель на тип "строка"; String variable = "Привет, это данные в формате строки"; // тип данных String пишется с большой буквы;

Отладка или "режим дебаггинга";
Это построчное выполнение программы для поиска ошибки или анализа, чтобы понимать, что происходит в коде;
Можно поставить слева от кода "точки остановки"; Запускается режим кнопкой с жуком, продолжается в консоли есть кнопка play или через контексное меню, когда правой кнопкой мыши нажимаешь на сам main class  файл в дереве файлов;

Функции;
Функция прописывается отдельно через public static void functionName(String variblesName)
Внутри самой функции передается значение;
